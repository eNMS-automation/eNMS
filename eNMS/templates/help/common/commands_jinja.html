Interpret Commands as Jinja2 Template

Select the checkbox to interpret the Commands text as a Jinja2 template. The output of the rendered template is used as the list of commands for execution. A common use case is looping over values to generate a command per value.

Example: Execute command for each interface

The following code issues a "show interface" command for each interface of interest: 

{{raw}}{% for intf in get_var("interfaces", device=device.name) -%}{{endraw}}
{{raw}}  show interface {{ intf }}{{endraw}}
{{raw}}{% endfor %}{{endraw}}

Note the minus sign on the closing Jinja2 tag -%> with the for loop. This prevents Jinja2 from generating unwanted blank lines. See whitespace control.
